
# 月历悬浮球（Calendar Float）

角色卡命定之诗与黄昏之歌的DLC项目

本项目把“世界时间”、“节庆资料”、“读物资料”和“玩家自定义日程”整合成一个可直接在酒馆界面中使用的悬浮月历，并支持：

- 月历视图与当月事件总览
- 临时 / 重复事件的新增、编辑、删除与归档
- 节庆提醒与进行中事件高亮
- 节庆关联读物展示

---

## 如何添加数据

### 文件结构

```text
public/assets/data/
├── README.md              # 本说明文档
├── events/*.txt/          # 节庆介绍
└── books/*.txt/           # 节庆相关读物
└── reminders/*.txt/       # 节庆提示
└── index.yaml             # 索引与映射表
```
### 内容说明

映射表放了触发词、日期、文件位置等等

reminders是D0深度提示词，会在events的X天前/进行时插入提醒LLM

[event]为节庆介绍及相关活动，会在满足条件(当前时间和地点)时触发

[book]文本支持MD格式，`[newpage]`会开新一页

当前部分节庆和读物正文仍是占位文本(咕咕咕)，但已经保留好对应条目与触发逻辑，后续只需要补正文就行

## 工作方式

本项目的月历系统主要依赖两部分：

### 1. 前端脚本层

负责：

- 渲染悬浮月历UI
- 读取mvu当前世界时间
- 管理玩家创建的日历事件
- 展示节庆、提醒与关联读物
- 节庆提醒
- 将仓库里的节庆、读物内容拼装成ejs条目，然后注入主世界书
- 安装、重装与卸载条目

### 2. 世界书ejs条目层

负责：

- event、book、mvu变量及规则会通过世界书插入提示词
- 依据时间、地点、聊天提及关键词触发对应内容到指定的插入位置及顺序
